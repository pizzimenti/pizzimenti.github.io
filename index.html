Cupcake Shop!

Project Objectives

  1. Add classes to style text
  2. Link email addresses
  3. Practice CSS selectors

This project was our introduction to styling text in large blocks that repeated in other places throughout our html document. We learned the differences between id and class and how to create a link to an email address.

We also applied our knowlege of specificity as it applies to CSS selectors in order to override instances of CSS that we do not want to apply to a specific case or section of a site.
